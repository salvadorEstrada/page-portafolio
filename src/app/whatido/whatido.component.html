Backend project structure

Backend technology

a) I developed this application adopting Spring Boot 2.6, and Java 8, for data persistence, I used Java Persistence API (JPA). b) Spring security and JSON web token (JWT). c) Mysql 8.0 version as database.

Frontend technology

I developed this frontend application utilizing Angular Framework version 13 and Angular Material library.
