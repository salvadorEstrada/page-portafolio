Overview about the project

Backend project structure

This application was developed with the purpose of evaluating the knowledge of a user in different areas of technology, which could be Languages Programming, Data Bases or any other area of knowledge.
a) The application managed corresponds to an administrator user. This user can do changes regarding the information contained in the application.
b) A normal user must be registered, once the user is registered will be able to receive a token to get the corresponding resources inside the application.
c) There are different Categories, Exams, and Questions in which the user could test his knowledge, every question has a limit of time.


Frontend project structure

This angular client was developed with the purpose of consuming the services from the backend.
b) The application contains a view with a panel for administrator users. In this view administrator user will be able to manage the application information such as create Categories, Exam tests, and Questions, and will be able to enable and disable this information.
c) Panel for the standard user, in this view user must be registered to get the application access, once the user gets login success can make knowledge challenges according to enable categories by the administrator.
