Overview about the project Exam test

Backend project structure

This application was developed with the porpuse of evaluate the knowledge of a user in different areas of technology, could be Languages Programming, Data Bases or any other area of knowledge.
a) The managed of application corresponds to an administrator user. This user can do changes regarding the information content in the application.
b) A normal user must be registered, once the user is registered will be abled to recieve a token to get the corresponding resources inside the application.
c) There are diferent Categories in which the user coul be test your knowledge evey question has a limit of time.


Frontend project structure

This angular client was developed with the porpuse of consume the services from backend.
b) The application contains a view with panel for administrator user. In this view administrator user will able to manage the application information such as create Categories, Exam test, Cuestions, will be abled to enabled and disabled this information.
c) Panel for standar user, in this view user must be registered to get the application access, once the user get login success can make knowledge challenges acording enabled categories by administrator.
